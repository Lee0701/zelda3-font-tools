font			0x70000 (CPU 0x0e8000)
font properties		0x73844 (CPU 0x0eb844)

포인터는 세번에 걸쳐 로드하며, 두곳에 하드코딩 되어 있음.
font properties read        0x774fb (CPU 0x0ef4fb)
font properties read 2      0x77cb2 (CPU 0x0efcb2)


게임 프로그램에서 계산된 각 글자의 시작 오프셋은 램(0x7f5b00~)에 포인터 테이블로 저장된다.
    print(f'{offset:04x}') 부분의 주석 처리를 해제하고 디코딩을 실행하면 값이 일치함.


대사 포인터 테이블은 따로 존재하지 않고 게임 실행시에 오프셋을 계산해서 생성한다. (0x1cf37a)
    생성된 테이블은 0x7f71c0에 위치.
